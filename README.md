# 💬 Let's Chat — P2P Real-Time Chat App

A real-time peer-to-peer chat application built with **Node.js**, **Express**, **Socket.IO**, **MongoDB**, and **EJS**.

---

## 🚀 Features

- Real-time messaging using Socket.IO
- Persistent chat history stored in MongoDB
- Automatic friend list management
- User search functionality
- Unique chatrooms per user pair
- ASCII art welcome screen via Figlet

---

## 🛠 Tech Stack

| Layer      | Technology              |
|------------|-------------------------|
| Backend    | Node.js, Express        |
| Realtime   | Socket.IO               |
| Database   | MongoDB + Mongoose      |
| Templating | EJS                     |
| Utilities  | Figlet, dotenv          |

## ⚙️ Setup & Installation

### 1. Clone the repository

```bash
git clone https://github.com/Yuv-Raj-15/ChatApp.git
cd lets-chat
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env` file in the root directory:

```env
MONGO_URL=mongodb://localhost:27017/letschat
PORT=3000
```

### 4. Start the server

```bash
node index.js
```

Visit `http://localhost:3000` in your browser.

## 🔌 API Routes

| Method | Route          | Description                        |
|--------|----------------|------------------------------------|
| GET    | `/`            | Home page with ASCII art welcome   |
| POST   | `/chats`       | Register/login user, show chat UI  |
| POST   | `/p2p`         | Open P2P chatroom between two users|
| GET    | `/search-user` | Search user by name (JSON)         |

---

## 🔧 Socket.IO Events

| Event          | Direction         | Description                        |
|----------------|-------------------|------------------------------------|
| `join-room`    | Client → Server   | Join a specific chatroom           |
| `send-msg`     | Client → Server   | Send a message to a room           |
| `received-msg` | Server → Client   | Broadcast message to room members  |

## 🗄️ MongoDB Models

### User (`userdb`)
```js
{ name: String, email: String }
```

### Chat (`chatdb`)
```js
{
  chatroom: String,  // sorted user IDs joined by "_"
  chats: [{ receiver: String, txt: String }]
}
```

### Userdata (`userdata`)
```js
{
  name: String,
  friends: [{ name: String, name_id: ObjectId }]
}
```

---

## 📝 Notes

- Chatroom IDs are generated by sorting two user IDs and joining them with `_`, ensuring the same room is used regardless of who initiates.
- Users are auto-created on first login — no password/auth system is included by default.
- Friends are automatically added to each user's list on first P2P chat.
